# Billboard Hit Analysis 

This project analyzes 60 years of Billboard Hot 100 data to uncover patterns behind successful songs and artists.

## Objectives

1. Identify artists that dominated the charts.
2. Understand how long songs stay on the chart.
3. Explore how music trends evolved across decades.
4. Investigate whether songs that start strong stay longer.
5. Build a model to predict hit songs.
6. Identify factors that influence chart success.

## Technologies Used

- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-Learn
- Jupyter Notebook


## Key Insights

- A small number of artists dominate the charts.
- Most songs leave the chart quickly while a few stay for long periods.
- Chart trends evolve across decades.
- Songs that reach higher ranks often remain longer on the chart.
